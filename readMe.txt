To Host site on computer:

If using Python 2:
    1. Go to directory where the server2.py script is and type: "python server2.py"
    2. Go to your web browser and in the url bar, type: "localhost:8000"

If using Python 3:
    1. Go to directory where the server3.py script is and type: "python server3.py"
    2. Go to your web browser and in the url bar, type: "localhost:8000"
    *Note*: I don't have python 3 so it may not work.....
